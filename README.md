# AES-GCM 数据包解密工具集

本项目提供了一套用于解密 AES-GCM 加密数据包的工具，包括命令行工具、Burp Suite 插件和 Yakit 插件。

## 背景

通过对某 Flutter Web 应用的 JavaScript 代码逆向分析，发现其使用 AES-GCM 算法加密 API 通信数据。本工具集用于自动检测并解密此类加密数据包。

## 加密参数

| 参数 | 值 |
|------|-----|
| 加密算法 | AES-GCM |
| 密钥 | `0123456789abcdef0123456789abcdef` |
| 密钥长度 | 32 字节 (UTF-8 编码) → AES-256 |
| Nonce 长度 | 12 字节 |
| Tag 长度 | 16 字节 |
| 数据格式 | Base64([12字节Nonce][加密数据][16字节Tag]) |

## 工具使用

### 1. 命令行工具 (decrypt_packet.py)

独立的 Python 脚本，用于手动解密数据包。

**依赖安装：**
```bash
pip install pycryptodome
```

**使用方法：**
```bash
# 测试模式
python3 decrypt_packet.py test

# 解密指定数据
python3 decrypt_packet.py "BASE64加密数据"
```

**示例：**
```bash
$ python3 decrypt_packet.py "7wPNx3PQI9fpvWgoJlbbIbAiK9EkQRVCrzlbPmTIH83BQua1TPePA1P5oJzyzT1WsFPkN2Ckwk1ufhLsAxyFlZiGnpUyqziS4v4="

解密成功!
----------------------------------------
{"username":"admin#","password":"admin!@#456"}
----------------------------------------
```

### 2. Burp Suite 插件

被动解密插件，自动检测并解密 HTTP 响应中的加密数据。

**安装步骤：**

1. 确保已安装 Jython 或配置 Python 环境
2. 安装依赖：`pip install pycryptodome`
3. Burp Suite → **Extender** → **Add**
4. Extension type: **Python**
5. 选择文件 `plugins/burp_aes_gcm_decryptor.py`

**功能特性：**

- 被动监听所有 HTTP 响应
- 自动识别 Base64 编码的 AES-GCM 加密数据
- 支持 AES-256-GCM 和 AES-128-GCM 两种模式
- 解密成功的请求会：
  - 高亮显示（绿色）
  - 添加注释 "AES-GCM Decrypted"
- 提供独立的 UI 面板：
  - 查看解密日志
  - 动态更新密钥

**界面说明：**

插件会在 Burp Suite 中添加一个名为 "AES-GCM Decryptor" 的标签页，显示所有解密记录。

### 3. Yakit 插件

被动解密插件，劫持 HTTP 响应并自动解密。

**安装步骤：**

1. Yakit → **插件** → **新建插件**
2. 将 `plugins/yakit_aes_gcm_decryptor.yak` 的内容粘贴进去
3. 保存并启用插件

**功能特性：**

- 劫持 HTTP 响应
- 自动检测并解密 AES-GCM 加密数据
- 解密成功后：
  - 自动替换响应体为明文 JSON
  - 添加响应头 `X-AES-GCM-Decrypted: true`
- 在日志中输出解密结果

**日志输出示例：**
```
============================================================
[+] AES-GCM 解密成功!
URL: https://api.example.com/api/user/login
------------------------------------------------------------
解密结果:
{
  "code": 0,
  "message": "success",
  "data": {...}
}
```

## 技术细节

### 加密流程

```
明文 → AES-GCM 加密 → [Nonce][密文][Tag] → Base64 编码 → 传输
```

### 解密流程

```
Base64 数据 → Base64 解码 → 提取 Nonce(前12字节) + Tag(后16字节) → AES-GCM 解密 → 明文
```

### 密钥来源

通过对 Flutter Web 编译的 JavaScript 文件 (`main.js`) 进行逆向分析，发现硬编码的加密密钥：
